# TV-4 Ping Pong
Example of a 4 Local Player Pong Type of Game

# Description
This is a student Pong version of the game which adds the ability to play with 4 players. It was developed during a whole week and its purpose is to learn how to code and make simple games with a twist, in Unity.

Players 1 and 2 uses the WASD keys to move, while the Players 3 and 4 uses the Arrow keys.

# Features
- Up to 4 Local Players in the same game.
- You can play it in the browser.

# Programs Used
- Unity 6
- C# (Visual Studio)

# Lessons
I've learned how to make simple web browser games with a twist.
